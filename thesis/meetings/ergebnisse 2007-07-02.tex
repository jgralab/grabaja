\begin{itemize}
	\item{Der Text zum Extraktor soll wie folgt überarbeitet werden:}
	\begin{itemize}
		\item{Fortsetzung gemäß dem Stand der Implementation.}
		\item{Es soll allgemein ein höherer Anteil an Technik erarbeitet werden.}
		\item{Überschriften sollen aussagekräftiger sein.}
		\item{Es soll auf mögliche Fehler und deren Behandlung eingegangen werden.}
		\item{Im Abschnitt zur Erzeugung / Voraussetzungen soll kurz auf die benötigte ANTLR-Version eingegangen werden.}
		\item{Im Abschnitt zur Erzeugung soll verdeutlicht werden, dass dieser Vorgang nur einmalig notwendig ist (und nicht bei jeder Verwendung o.ä.).}
		\item{Im Abschnitt zur Erzeugung soll eine Abbildung zur Verzeichnisstruktur im lokalen Repository eingefügt und sich im Text entsprechend darauf bezogen werden.}
		\item{Der Abschnitt über den Ablauf soll mittels UML-Aktivitätsdiagramm(en) verdeutlicht werden.}
		\item{Der Abschnitt zur Verwendung soll in Englisch vorliegen (zwecks Einbeziehung in das ebenfalls rein englische GuPro-Handbuch).}
		\item{Listings, Dateinamen und ähnliches soll in einer Typewriter-Schrift mit Blocksatz formatiert werden.}
		\item{Weitere Punkte siehe Notizen und Anmerkungen in der ausgedruckten Fassung.}
	\end{itemize}
	\item{Es soll überprüft werden, ob es möglich ist, für die Analyse und den Reflection-Mechanismus externer (vom analysierten Code verwendeter) Klassen einen anderen ClassPath als für den Extraktor selbst zu verwenden. Dies ist z.B. dann sinnvoll, wenn gleiche Klassen in unterschiedlicher Version zum Einsatz kommen / benötigt werden. Falls dies möglich ist, soll ein kurze Abschätzung hinsichtlich des Aufwands einer Implementation erfolgen.}
	\item{Das verwendete Schema soll in der jetzigen Struktur verbleiben und nicht auf die Package-Struktur des Metamodells umgestellt werden (da JGraLab / TGSchema2Java dies nicht unterstützen).}
	\item{\emph{"`fullQualifiedName"'} soll in \emph{"`fullyQualifiedName"'} geändert werden.}
	\item{Die Schätzung der benötigten Kanten / Knoten soll eventuell über eine Abschätzung der Codezeilen über die Dateigröße erfolgen.}
	\item{Nur temporär während der Erzeugung des Graphen verwendete Kantenklassen sollen von einer abstrakten Klasse \emph{"`TemporaryExtractorEdge"'} o.ä. abgeleitet werden.}
  \item{Das nächste Treffen findet am 17. Juli 2007 um 11:00 s.t. in B108 statt.}
  \item{Einreichung der Dokumente bis 16. Juli 2007 (morgens).}
  \end{itemize}
